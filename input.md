## System overview

The project is a distributed and fault-tolerant seismic analysis platform. It receives real-time seismic measurements from a provided simulator, redistributes them to replicated processing services, performs frequency-domain analysis, classifies detected events, and stores them in a shared persistence layer with duplicate-safe behavior.

A real-time dashboard is provided to monitor detected events and inspect historical information.
